# Nigerian Tax Hub

A comprehensive tax management platform for Nigerian citizens and corporations.

## Features
- Personal Income Tax Calculator
- Corporate Tax Analysis with AI
- Tax Filing System
- Payment Integration
- Document Management

## Getting Started
```bash
# Install dependencies
pnpm install

# Setup database
pnpm db:generate
pnpm db:push

# Run development server
pnpm dev
```

## Tech Stack
- Next.js 14
- TypeScript
- Prisma
- PostgreSQL
- OpenAI
- Tailwind CSS
